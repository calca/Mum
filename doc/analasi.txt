=== Struttura web-gallery

ogni album deve essere contenuto in una direttory differente cosi' realizzata

/index[|N].html sono la lista di tutte le immagine presenti nella
		gallery
/thumb/         sono le immagine di grandezza 120x80 ??
/size-$size/    cartella contetne le immagini di dimensione
		$size={640x480 , 800x600 , 1024x768 }
/html/          cartella contente i file html in fomrato nome_immagine.html

sul web viene creata una nuova directory di nome $galleria decisa
dall'utente e viene, se desiderato dall'utente. aggiornato il file
index.html della root
 
Gli index delle gallery sono formati da tabelle 4x4 contente i thumb
delle immagine.

L'upload avviene tramite ftp

+ controllare che l'utente quando l'album non vada gia a sovrascrivere
un album gia' presente
+ ordinare l'album in base all'ordinamento presente nella dir
+ controllo che non esiste gia' un album sul ftp con quel nome
+ estrazioni dati delle immagini jpeg con la lib exif
